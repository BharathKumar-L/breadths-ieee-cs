QUIZ

This engaging and intellectually stimulating competition is designed to challenge your knowledge, quick thinking, and teamwork. Participants will navigate through a series of rounds, each with unique formats and themes, testing their expertise across a broad range of topics. Whether you're a seasoned quizzer or a curious mind, this quiz promises to be an exciting journey filled with learning.


Duration: 80 minutes (1 hour 10 minutes)

Prizes: Top 3 teams will be awarded with cash prizes

EVENT DETAILS

Preliminary Round: Online Quiz (if necessary)

Format: Online Multiple Choice Questions (MCQs)

Description: If the number of participants exceeds the limit, an online quiz will be conducted. Participants will answer a set of MCQs covering a wide range of general knowledge and technical topics.

Duration: 30 minutes

Selection Criteria: The top-scoring teams will be selected to participate in the on-site rounds.

Instructions: Participants will receive a link to the quiz and must complete it within the specified time.



On-Site Rounds (for selected participants)

Round 1: CONNEXIONS

Format: Visual Identification

Description: Participants will be shown images related to technological innovations. Teams will have to identify the images and provide the answers.

Scoring: Points for correct identification and no negative marking.

Round 2: Written Quiz (Pen and Paper)

Format: Single Word Answer Questions

Description: In this round, participants will answer a set of questions provided to them in a pen-and-paper format.

Instructions: Each team will receive a question paper and answer sheet. They will have 30 minutes to complete the quiz.

Scoring: Points for each correct answer. Accuracy is key, as each response should be a single word.

Round 3: Rapid Fire

Format: Quick Response Questions

Description: In this high-energy round, teams will answer as many questions as possible within a short time frame. Questions will cover a wide range of topics and require quick thinking and rapid responses.

Scoring: Points for each correct answer. Incorrect answers do not incur penalties, but time is critical.

Round 4: Final Showdown

Format: Quick Response Round

Description: Teams will compete to respond first to each question. The team that responds first will have their answer considered.

Instructions: Teams must raise their hands or use a designated signal to indicate they are ready to answer. If the answer is correct, they will earn points; if the answer is incorrect, negative marking will be applied.

Scoring: Points for each correct answer. Negative points for incorrect answers.

INSTRUCTIONS

  • Each team should consist of 1-3 members.
  • Participants must maintain decorum and respect the event coordinators and other participants.
  • Using mobile phone is strictly prohibited.
  • Any form of cheating or malpractice will lead to immediate disqualification.
  • The decisions of the Quiz Masters are final and binding.
  • Certificates will be provided to all participants.

REPLICA RUMBLE

The Web Recreation Challenge aims to test participants web development skills by challenging them to recreate a web page from scratch using HTML,CSS and JavaScript(optional). This event is designed to assess the participants' ability to analyze a given design, understand its structure, and implement it accurately and efficiently.

EVENT DETAILS

Participants will be divided into two sets, with each set working on the same task but at different times. The event will proceed as follows:


  1. Phase 1:

    • A fully designed web page will be shown to all participants. This page will serve as the template they need to recreate.
    • Participants will have a brief time to observe the page and analyze its design elements, structure, and functionality.

  2. Phase 2:

    • The first set of participants will be given a fixed amount of time to recreate the web page shown to them. They will work individually to code the HTML, CSS, and JavaScript(optional) necessary to match the design as closely as possible.
    • Once the time is up, their work will be collected and evaluated.

  3. Phase 3:

    • The second set of participants will then have their turn to recreate the same web page under the same conditions.
    • Similarly, their work will be collected and evaluated at the end of the allotted time.

  4. Phase 4:

    • All submissions from both sets of participants will be evaluated together based on criteria such as design accuracy, code efficiency, responsiveness, and adherence to best practices.
    • Judges will select the top performers from the entire pool of participants, regardless of which set they were in.

INSTRUCTIONS

General Rules:

  • Individual Participation: The contest is an individual event. Each participant must work alone.
  • Language Restrictions: Only HTML and CSS are allowed. Use of JavaScript is optional and use of frameworks are not allowed.
  • Original Work: All work must be done on-site during the contest period. Pre-written code or templates are not allowed.
  • Submission: All submissions must be handed in by the deadline. Late submissions will not be accepted.

Technical Rules:

  • Code Quality: The code should be clean, well-organized, and properly commented.
  • Compatibility: The recreated webpage should be compatible with the latest versions of major browsers (Chrome, Firefox, Edge).
  • Responsiveness: While responsiveness is not the primary focus, the webpage should render well on standard desktop screen sizes.

JUDGING CRITERIA

  • Accuracy: How closely the recreated webpage matches the original in terms of layout, design, and functionality.
  • Code Quality: Cleanliness, organization, and comments in code.
  • Creativity: Any enhancements or unique elements added by the participant that improve the overall design.
  • Efficiency: Efficient use of your code to achieve the design with minimal code redundancy.

DISQUALIFICATION

  • Plagiarism: Any participant found copying code from external sources or other participants will be disqualified.
  • Misconduct: Any form of misconduct, including but not limited to cheating, disrupting others, or violating contest rules, will lead to disqualification.
  • Non-Compliance: Failure to comply with the submission guidelines will result in disqualification.

Circuit Debugging

  • Each team receives a circuit board with pre-introduced faults.
  • The faults may include open circuits, short circuits, incorrect component values, or misconnected components.
  • Based on the judging criteria, 5 teams will be selected for the final round.

Rules & Regulations

  • 2 participants per team.
  • The competition consists of two rounds: preliminary round and the final round.
  • Judging criteria include the time taken to solve the error, accuracy, and explanation of the debugged circuit.
  • Ensure that you don’t damage the components; a multimeter will be provided.
  • All participants will be provided with a certificate.
